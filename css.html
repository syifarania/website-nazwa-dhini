PENGERTIAN CSS

CSS (Cascading Style Sheets) adalah bahasa pemrograman yang digunakan untuk mendesain dan mengatur tata letak (layout) serta tampilan visual elemen-elemen dalam sebuah halaman web

FUNGSI CSS

Fungsi CSS adalah untuk mengatur tampilan dan layout elemen-elemen dalam halaman web, memisahkan desain dari struktur HTML. Dengan CSS, kita bisa mengubah warna, ukuran, posisi elemen, serta menambahkan animasi dan efek visual

SEJARAH CSS

1994: CSS pertama kali diperkenalkan oleh Hakon Wium Lie dan Bert Bos sebagai cara untuk memisahkan desain dari struktur HTML. 1996: CSS1 dirilis oleh W3C, menetapkan dasar pengaturan gaya seperti warna, font, dan layout. 1998: CSS2 dirilis, menambahkan fitur seperti posisi elemen dan media queries untuk desain responsif. 2004-2011: CSS2.1 diperkenalkan untuk memperbaiki kompatibilitas antar browser. 2008-sekarang: CSS3 dirilis dengan fitur baru seperti animasi, transisi, dan layout responsif (Flexbox, Grid).

KELEBIHAN CSS

Pemisahan Konten dan Desain: Memudahkan pemeliharaan kode dan pembaruan desain tanpa mengubah HTML. Desain Responsif: Dapat membuat halaman web menyesuaikan tampilan pada berbagai perangkat menggunakan media queries. Efek Animasi: Mendukung animasi dan transisi tanpa memerlukan JavaScript. Kinerja yang Cepat: Mengurangi beban server karena desain ditangani di sisi klien (browser).
